首页

获取会场信息

  • 请求路径
GET /v0/meeting/primary  请求主会场
GET ／v0/meeting／secondary  请求分会场

<!--补充说明-->
两者的返回消息一致
  • 返回消息
{
    status: '',
    data: [
        {
          id: '', //活动id，唯一
          avatar: '', // 照片url
          name : '',  // 票的名称
          beginedAt: '',  //开始时间
          limitNum: '', // 票的限制数量
          restNum: '' // 剩余票的数量
          price: '', 票的价格
        }
    ]
}

回顾

视频回顾

  • 请求路径

    GET /v0/history/video

  • 返回消息

    {
  status: '',
  data: [
      {
        videoUrl: '', //视频回顾中视频的URL
        desc : '', // 介绍说明
        placeType: '', 地点类型，1表示主会场，2表示分会场。
      }
  ]
}

图文回顾

GET /v0/history/graphic
  • 返回消息
    {
  status: '',
  data: [
      {
        avatar: '', // 图文回顾中图片url
        desc : '', // 介绍说明
        updatedAt: '', //仅在图文回顾中存在，显示时间
        viewedNum: '', // 浏览次数，仅在图文回顾中出现
      }
  ]
}
